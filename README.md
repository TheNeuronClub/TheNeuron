
### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/TheNeuronClub/TheNeuron.git
   ```
2. change directory to TheNeuronClub 
   ```sh
   cd TheNeuron
   ```   
3. Install NPM packages
   ```sh
   npm install
   ```
4. run the project
   ```sh
   npm run dev
   ```   
   
Preview on Local system : http://localhost:3000/
